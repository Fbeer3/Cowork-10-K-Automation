# 10-K Financial Analysis Dashboard Project

Turning 10-K financial statement data into interactive dashboards and executive-level insights.

---

## Key Insights

- **Microsoft:** AI-driven growth is strong, but rising capex is compressing free cash flow  
- **Apple:** High margins with more mature revenue growth  
- **Amazon:** Margin expansion driven by AWS profitability  
- **Meta:** Advertising recovery driving strong top-line growth
